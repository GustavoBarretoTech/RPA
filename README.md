# 🤖 Automações RPA (Python)

Este repositório contém scripts desenvolvidos para automatizar tarefas repetitivas no sistema operacional e no navegador.

## 🚀 Funcionalidades

As ferramentas inclusas aqui realizam:

*   **Navegação Web Automática:** Scripts que acessam o navegador, buscam URLs específicas e interagem com elementos.
*   **Interação com SO:** Automação de tarefas direto no sistema operacional (manipulação de janelas e arquivos).
*   **Web Scraping:** Extração e coleta de dados básicos de páginas HTML.
*   **Organizador de Arquivos:** Script para tratamento e organização automática de arquivos em pastas com base em suas extensões.

## 🛠️ Tecnologias Comuns
*   Python
*   Bibliotecas sugeridas: `Selenium`, `PyAutoGUI`, `OS`, `BeautifulSoup`, `Shutil`
